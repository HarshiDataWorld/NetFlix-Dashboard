# NetFlix-Dashboard
Power BI dashboard analyzing Netflix movies and TV shows to uncover trends in content type, genres, ratings, countries, and release years. Built using a publicly available dataset.
What your dashboard does

Movies vs TV Shows comparison

Ratings distribution by category

Top countries by content count

Trends over time in content releases

Interactive filters (year, country, type)

This is similar to what many Netflix dashboards do, as seen in community submissions and GitHub repos like Netflix Analytics dashboards.
